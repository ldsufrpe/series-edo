Solução 1.5.10.1.

Devemos encontrar o intervalo de convergência. Usamos o teste de razão, que é muito confiável para esse fim:

\begin{equation*} \begin{aligned} \lim_{n\to\infty} \left| \frac{ \left( \frac{ (8 x)^{n+1} }{ (n+1)^{3} } \right) } { \left( \frac{ (8 x)^n }{ n^{3} } \right) } \right| = \lim_{n\to\infty} \left| \frac{ 8 x }{ \left( \frac{n+1}{n} \right)^{3} } \right| = \left| 8 x \right| \end{aligned} \end{equation*}

Em seguida, usamos a expressão resultante para obter o intervalo de convergência verificando quando é menor que 1:

\begin{equation*} \left| 8 x \right| \lt 1 \iff \left| x \right| \lt \frac 1 {8} \iff \frac{-1}{8} \lt x \lt \frac 1 {8} \end{equation*}

Assim, devemos simplesmente verificar os pontos das extremidades.

Quando \(x = \frac{-1}{8}\text{:}\)

\begin{equation*} \sum_{n=1}^\infty \frac{(8 x)^n}{n^{3}} = \sum_{n=1}^\infty \frac{(-1)^n}{n^{3}} \end{equation*}

Isso é convergente pelo teste da série alternada, uma vez que claramente \(\frac 1 {n^{3}}\) tende a zero \(0\text{.}\)

quando \(x = \frac 1 {8}\text{:}\)

\begin{equation*} \sum_{n=1}^\infty \frac{(8 x)^n}{n^{3}} = \sum_{n=1}^\infty \frac 1 {n^{3}} \end{equation*}

Isso é convergente como uma p-série.
