Tente primeiro \(y=\frac{1}{c+x^2}\text{.}\)

Então \(y^\prime = \frac{-4x}{(c+x^2)^2}\) enquanto

\(y-y^2 = \frac{1}{c+x^2} - \frac{1}{(c+x^2)^2}= \frac{c+x^2}{(c+x^2)^2} - \frac{1}{(c+x^2)^2}=\frac{c+x^2-1}{(c+x^2)^2}\)

Assim \(y^\prime\neq y-y^2\) para esta função, não é a solução.

Agora tente \(y=\frac{1}{1+ce^{-x}}\text{.}\) Então

\(y^\prime = \frac{ce^{-x}}{1+ce^{-x}}\)

enquanto

\(y-y^2=\frac{1}{1+ce^{-x}}-\frac{1}{(1+ce^{-x})^2}=\frac{1+ce^{-x}}{(1+ce^{-x})^2}-\frac{1}{(1+ce^{-x})^2}=\frac{ce^{-x}}{(1+ce^{-x})^2}\)

Então esta função é a solução

Definindo \(y(0)=4\) temos

\(4 = \frac{1}{1+c}\)

Resolvendo para c, obtemos \(c = \frac{1}{4}-1=-0.75\text{.}\)

Então a solução para o Problema de Valor Inicial é \(y = \frac{1}{1 - 0.75e^{-x}}\)
