This is a separable differential equation and we can solve it explicitly. We shall do so shortly. See Example Exemplo 1.20, below. But, before doing that, we'll see what we can learn about the behaviour of solutions to differential equations like this without finding formulae for the solutions. It turns out that we can learn a lot just by watching the sign of \(P'(t)\text{.}\) For concreteness, we'll look at solutions of the differential equation

\begin{gather*} \diff{P}{t}(t)=\big(\,6000-3P(t)\,\big)\,P(t) \end{gather*}

We'll sketch the graphs of four functions \(P(t)\) that obey this equation.

  • For the first function, \(P(0)=0\text{.}\)

  • For the second function, \(P(0)=1000\text{.}\)

  • For the third function, \(P(0)=2000\text{.}\)

  • For the fourth function, \(P(0)=3000\text{.}\)
