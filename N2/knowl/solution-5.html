Solução 1.3.11.1.

\begin{align*} a_N \amp=S_N-S_{N-1}=\left[(-1)^N+\dfrac{1}{N}\right]-\left[(-1)^{N-1}+\dfrac{1}{N-1}\right]\\ \amp=(-1)^N-(-1)^{N-1} + \frac{1}{N}-\frac{1}{N-1}\\ \amp=(-1)^N+(-1)^{N} + \frac{N-1}{N(N-1)}-\frac{N}{N(N-1)}\\ \amp=2(-1)^{N} -\frac{1}{N(N-1)}\\ \end{align*}

Observe, no entanto, que \(a_N\) é só igual \(S_N-S_{N-1}\) quando \(N \geq 2\text{:}\) caso contrário, estamos tentando calcular \(S_1-S_0\text{,}\) mas \(S_0\) não está definida. Então, nós achamos \(a_1\) separadamente:

\begin{align*} a_1\amp=S_1=(-1)^1+\frac{1}{1}=0\\ \end{align*}

Juntando tudo:

\begin{align*} a_n\amp=\begin{cases} 0 \amp \mbox{ se } n=1\\ 2(-1)^{n} -\frac{1}{n(n-1)} \amp\mbox{ caso contrário } \end{cases} \end{align*}
