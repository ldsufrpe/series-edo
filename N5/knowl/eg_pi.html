Exemplo 1.10. Calculando o número \(\pi\).

Existem vários métodos de calcular \(\pi\) para qualquer grau de precisão desejado. Muitos deles usam a série de Maclaurin

\begin{align*} \arctan x \amp= \sum_{n=0}^\infty (-1)^n\frac{x^{2n+1}}{2n+1} \end{align*}

doTeorema 1.7. Uma vez que \(\arctan(1)=\frac{\pi}{4}\text{,}\) a série nos fornece uma fórmula para \(\pi\text{:}\)

\begin{align*} \frac{\pi}{4} = \arctan 1 \amp= \sum_{n=0}^\infty \frac{(-1)^n}{2n+1}\\ \pi \amp= 4 \left( 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots \right) \end{align*}

De fato, esta série não é muito útil para calcular \(\pi\)pois converge lentamente. Se aproximamos s série pela sua \(N\)-ésima soma parcial, então o teste da série alternada nos diz que o erro é limitado pelo primeiro termo que eliminamos. Para garantir que temos pelo menos 2 dígitos decimais corretos de \(\pi\) corretos, preciamos somar os 200 primeiros termos!

Uma maneira muito melhor de calcular \(\pi\) usar esta série é usar a igualdade: \(\tan\frac{\pi}{6}=\frac{1}{\sqrt{3}}\text{:}\)

\begin{align*} \pi \amp= 6\arctan\Big(\frac{1}{\sqrt{3}}\Big) = 6\sum_{n=0}^\infty (-1)^n\frac{1}{2n+1}\ \frac{1}{{(\sqrt{3})}^{2n+1}}\\ \amp= 2\sqrt{3} \sum_{n=0}^\infty (-1)^n\frac{1}{2n+1}\ \frac{1}{3^n}\\ \amp=2\sqrt{3}\Big(1-\frac{1}{3\times 3}+\frac{1}{5\times 9}-\frac{1}{7\times 27} +\frac{1}{9\times 81}-\frac{1}{11\times 243}+\cdots\Big) \end{align*}

Novamente, esta é uma série alternada e então o erro que introduzimos ao truncá-la é limitado pelo primeiro termo eliminado. Por exemplo, se mantivermos dez termos, parando em \(n=9\text{,}\) temos \(\pi=3.141591\) (até 6 casas decimais) com um erro entre zero e

\begin{equation*} \frac{2\sqrt{3}}{21\times 3^{10}} \lt 3\times 10^{-6} \end{equation*}
