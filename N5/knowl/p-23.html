Para fazer isso, precisamos limitar o tamanho de \(e^c\text{,}\) e então considere o que acontece se \(x\) é positivo ou negativo.

  • Se \(x \lt 0\) então \(x \leq c \leq 0\) e portanto \(e^x \leq e^c \leq e^0=1\text{.}\)

  • Por outro lado, se \(x\geq 0\) então \(0\leq c \leq x\) e assim \(1=e^0 \leq e^c \leq e^x\text{.}\)

Em qualquer dos casos temos que \(0 \leq e^c \leq 1+e^x\text{.}\) Logo, o termo que representa o erro satisfaz

\begin{equation*} |E_n(x)|=\Big|\frac{e^c}{(n+1)!}x^{n+1}\Big| \le [e^x+1]\frac{|x|^{n+1}}{(n+1)!} \end{equation*}

Afirmamos que esse é o limite superior e, e portanro o erro \(E_n(x)\text{,}\) tende rapidamente para zero quando \(n \to \infty\text{.}\)
